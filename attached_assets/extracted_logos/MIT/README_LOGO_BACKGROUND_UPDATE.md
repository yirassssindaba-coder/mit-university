- `UPDATED_PARAGRAPH_TO_APPLY_ID.txt`: paragraf final untuk instruksi/perubahan website.
- `assets/logos/`: logo/user-supplied logo + notes penggunaan.
- `assets/campus-images/original/`: gambar kampus versi asli dari paket sebelumnya.
- `assets/campus-images/web-optimized/`: gambar kampus versi web.
- `assets/campus-images/thumbnails/`: thumbnail.
- `assets/backgrounds/hero_background_campus_overlay_1920x1080.jpg`: latar hero siap pakai.
- `assets/backgrounds/hero_background_css_snippet.css`: snippet CSS untuk memasang background.
- `assets/source-credits/`: catatan sumber gambar.
